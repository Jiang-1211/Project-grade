# Calendar
## 利用Netbean作為IDE，自己看書學習Java並且使用基本的流程控制和函數寫出的第一個視窗程式，讓使用者輸入年分，顯示出該年每個月的日期與對應的星期，並有增減年份的按鈕可以快速加減，不用重新輸入。  
  
## 大致步驟如下:  
1.將需要的JButton與JTable拖出來  
2.簡單設計一下排版  
3.將需要重複使用的計算過程寫成函數  
4.把主要的運算過程寫在主程式  
5.完成主程式後檢查最終的排版
